# Security

If you discover a security vulnerability in Dida, please report it privately
to **security@didabit.com**. Do not open a public issue for security reports.

We will acknowledge your report within a few business days. Please include
steps to reproduce and the affected version (Help > About).

Vulnerabilities in the underlying Code - OSS platform should additionally be
reported to Microsoft following the process at
<https://github.com/microsoft/vscode/blob/main/SECURITY.md>.
